# Customer-Ready Sales Playbook (CompliCore)

This playbook turns the current demo into a sellable service offer with clear packaging, proof, onboarding, and delivery standards.

---

## 1) Define the productized offer (what you sell)

### Core promise
**"We install AI operations that increase revenue and reduce owner workload in 7 days."**

### Offer ladder (simple, buyer-friendly)
1. **Audit Sprint (One-time):** $750–$2,500
   - Discovery + KPI baseline + opportunity map
   - Deliverable: 14-day action plan
2. **Implementation Week (One-time):** $2,500–$8,000
   - Setup dashboard, workflows, approvals, automation stack
   - Deliverable: live system + launch report
3. **Optimization Retainer (Monthly):** $1,000–$5,000/mo
   - KPI reviews, prompt tuning, automation expansion
   - Deliverable: weekly report + monthly roadmap

### ICP (Ideal Customer Profile)
- Property managers / STR operators with **10–200 units**
- Existing booking/revenue flow but inconsistent operations
- Team capacity bottlenecked by repetitive coordination

---

## 2) Make trust visible (what buyers need before paying)

### Required proof assets
- **2–3 case studies** (before/after metrics)
- **Live demo video** (3–5 minutes)
- **Sample weekly report** and **handoff package**
- **Security + reliability one-pager**

### Minimum numbers to show in sales calls
- Time saved per week
- Revenue impact (or conversion/occupancy proxy)
- Error reduction / missed follow-up reduction

---

## 3) Productize the buying motion (how deals close)

### Sales pipeline stages
1. Lead captured
2. Qualified (ICP + budget + urgency)
3. Demo completed
4. Audit sold
5. Implementation closed
6. Retainer expansion

### Call structure (30 min)
- 5m: current operations + bottlenecks
- 10m: live dashboard walkthrough
- 10m: ROI model (their numbers)
- 5m: close to next step (audit sprint)

### Proposal format (must be fixed-template)
- Scope in/out
- 7-day delivery timeline
- KPI baseline + success criteria
- Approval dependencies
- Price + payment terms

---

## 4) Harden delivery for paying customers

### Delivery SLAs
- Response time: < 1 business day
- Critical workflow issue triage: < 4 business hours
- Weekly reporting cadence: fixed weekday/time

### Operational safeguards
- Human approval gates for customer-facing actions
- Rollback path for each workflow
- Incident log + postmortem template
- Staging/demo workspace before production workspace

### Customer onboarding standard
- Single customer POC assigned
- Access checklist completed
- Baseline KPI card frozen before launch
- QA launch checklist signed off

---

## 5) Packaging + pricing page readiness

### What your website/sales doc must include
- Clear outcome statement
- Exactly who it is for / not for
- 3 package tiers with deliverables
- FAQ: setup time, security, required access, cancellation
- Primary CTA: **Book Audit Sprint**

### Pricing principles
- Lead with outcome, not hourly work
- Use implementation fee + recurring optimization
- Keep discounting rules explicit and rare

---

## 6) “Ready to sell” Definition of Done

Use this as the go-live gate for commercialization:

- [ ] Offer ladder and prices finalized
- [ ] Demo flow rehearsed and recorded
- [ ] Proposal + MSA/SOW templates prepared
- [ ] Onboarding email + intake form + access checklist finalized
- [ ] 2+ case studies with concrete metrics published
- [ ] Weekly report template and KPI scorecard used with at least one pilot
- [ ] Payment flow live (invoice link or checkout)
- [ ] Basic CRM pipeline tracking active
- [ ] Referral + testimonial ask process documented
- [ ] 30-day KPI targets defined per package

---

## 7) First 30 days execution plan

### Week 1: Proof + positioning
- Finalize offer, positioning, and ICP
- Record demo and publish one-pager
- Build outbound list of 50 qualified leads

### Week 2: Sell audit sprint
- Run 10–15 discovery calls
- Close first 2–3 paid audits
- Capture baseline metrics rigorously

### Week 3: Deliver implementation wins
- Launch first implementation clients
- Produce before/after snapshots
- Ask for testimonial immediately after first visible win

### Week 4: Retainer conversion
- Present optimization backlog
- Convert implementation clients to monthly retainers
- Publish first case study and start referral asks

---
