# Client Delivery Pack (Copy Per Client)

Create a copy of this folder for each client and complete all items before closeout.

## Required handoff artifacts

- Access map (tools + owners)
- SOP index
- Prompt vault index
- Automation map
- KPI dashboard link + KPI definitions
- Baseline vs Day-7 KPI snapshot
- 30-day optimization backlog

## Recommended folder structure

- `01-intake/`
- `02-discovery/`
- `03-build/`
- `04-qa/`
- `05-reporting/`
- `06-handoff/`
